The official promotional landing page for **CodeStorm 2026: Python Pursuit**, an intra-college coding competition organized by the Dept of Artificial Intelligence & Cyber Security at ICET. 

Designed with a modern "Deep Indigo Glass" aesthetic, this project focuses on high-performance CSS animations, mobile-first responsiveness, and a fully automated vanilla JavaScript frontend.

## ✨ Key Features

* **Glassmorphism UI:** Translucent, blurred-background cards and navigation optimized for both desktop and mobile.
* **Interactive Particle Engine:** Custom-built HTML5 `<canvas>` particle background that runs seamlessly without heavy external libraries.
* **Flawless Mobile Scrolling:** Engineered with a CSS pseudo-element (`body::before`) background fix to prevent the notorious URL-bar scroll glitch on mobile browsers.
* **Dynamic Components:**
  * Auto-calculating real-time countdown timer.
  * Butter-smooth "Pro Tips" rotation engine with a synchronized CSS keyframe loading bar.
  * Synchronized typewriter effect for the hero section.
* **Zero Dependencies:** Built entirely with raw HTML, CSS, and Vanilla JavaScript. No bulky frameworks.

## 📂 Sections Included

## 🛠️ Usage / Installation
Since this is a static website, no build processes or installations are required.
1. Clone the repository: `git clone https://github.com/yourusername/codestorm-2026.git`
2. Open the directory.
3. Double click `index.html` to run it in your browser.
